# Project01

Program to read files produced by a high-throughput sequencer, machine that
takes DNA and produces a file containing a sequence of nucleotides, and
calculate

- GC-content
- AT-content
- G count
- C count
- A count
- T count
- Sum count
- Total count
- Seq length
- AT/GC ratio
- GC classification
